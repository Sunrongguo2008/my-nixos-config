# NixOS 配置库

Welcome to my NixOS configuration repository! This repository contains my personal configurations and setups for NixOS, a powerful Linux distribution based on the Nix package manager.

欢迎来到我的 NixOS 配置仓库！本仓库包含我个人对 NixOS 的配置和设置，NixOS 是一个功能强大的 Linux 发行版，基于 Nix 包管理器。

Feel free to explore the configuration files, modify them, or use them as references for your own NixOS setup.

您可以随意浏览配置文件、修改它们，或者将它们用作您自己的 NixOS 设置的参考。
