## 支持的布局

mangowm 支持多种布局，可以为每个标签分配。

- `tile`
- `scroller`
- `monocle`
- `grid`
- `deck`
- `center_tile`
- `vertical_tile`
- `right_tile`
- `vertical_scroller`
- `vertical_grid`
- `vertical_deck`
- `tgmix`

## Scroller 布局

Scroller 布局将窗口定位在可滚动的条带中，类似于 PaperWM。

### 配置

| 设置                                 | 默认值        | 描述                                                                                     |
| :----------------------------------- | :------------ | :--------------------------------------------------------------------------------------- |
| `scroller_structs`                   | `20`          | 当窗口比例为 1 时两侧保留的宽度。                                                        |
| `scroller_default_proportion`        | `0.9`         | 新窗口的默认宽度比例。                                                                   |
| `scroller_focus_center`              | `0`           | 始终居中聚焦的窗口（1 = 启用）。                                                         |
| `scroller_prefer_center`             | `0`           | 仅当聚焦窗口在视图外时居中。                                                             |
| `scroller_prefer_overspread`         | `1`           | 当有额外空间时允许窗口过度扩展。                                                         |
| `edge_scroller_pointer_focus`        | `1`           | 即使窗口部分离屏也聚焦窗口。                                                             |
| `scroller_proportion_preset`         | `0.5,0.8,1.0` | 用于循环窗口宽度的预设。                                                                 |
| `scroller_ignore_proportion_single`  | `1`           | 忽略单个窗口的比例调整。                                                                 |
| `scroller_default_proportion_single` | `1.0`         | Scroller 中单个窗口的默认比例。**需要 `scroller_ignore_proportion_single=0` 才能生效。** |

> **警告：** `scroller_prefer_overspread`、`scroller_focus_center` 和 `scroller_prefer_center` 相互作用。它们的优先级顺序为：
>
> **scroller_prefer_overspread > scroller_focus_center > scroller_prefer_center**
>
> 要确保较低优先级的设置生效，你必须将所有较高优先级的选项设置为 `0`。

```ini
# Scroller 配置示例
scroller_structs=20
scroller_default_proportion=0.9
scroller_focus_center=0
scroller_prefer_center=0
scroller_prefer_overspread=1
edge_scroller_pointer_focus=1
scroller_default_proportion_single=1.0
scroller_proportion_preset=0.5,0.8,1.0
```

## Master-Stack 布局

这些设置适用于 `tile` 和 `center_tile` 等布局。

| 设置                       | 默认值 | 描述                                                  |
| :------------------------- | :----- | :---------------------------------------------------- |
| `new_is_master`            | `1`    | 新窗口成为 master 窗口。                              |
| `default_mfact`            | `0.55` | master 和 stack 区域之间的分割比例。                  |
| `default_nmaster`          | `1`    | 允许的 master 窗口数量。                              |
| `smartgaps`                | `0`    | 当只有一个窗口存在时禁用间隙。                        |
| `center_master_overspread` | `0`    | （Center Tile）如果没有 stack 存在，master 跨越屏幕。 |
| `center_when_single_stack` | `1`    | （Center Tile）当只有一个 stack 窗口时居中 master。   |

```ini
# Master-stack 配置示例
new_is_master=1
smartgaps=0
default_mfact=0.55
default_nmaster=1
```

## 切换布局

你可以动态切换布局，或使用 [标签规则](/docs/window-management/rules#tag-rules) 为特定标签设置默认布局。

**键位绑定示例：**

```ini
# 循环遍历布局
bind=SUPER,n,switch_layout

# 设置特定布局
bind=SUPER,t,setlayout,tile
bind=SUPER,s,setlayout,scroller
```
