现在你已经安装了 mangowm，让我们来设置你的环境。

## 初始设置

1. **创建配置目录**

   mangowm 会在 `~/.config/mango/` 中查找配置文件。

   ```bash
   mkdir -p ~/.config/mango
   ```

2. **复制默认配置**

   在 `/etc/mango/config.conf` 提供了一个默认配置文件。将其复制到你的本地目录以开始自定义。

   ```bash
   cp /etc/mango/config.conf ~/.config/mango/config.conf
   ```

3. **启动 mangowm**

   你现在可以从 TTY 启动合成器。

   ```bash
   mango
   ```

   可选：指定自定义配置文件路径：

   ```bash
   mango -c /path/to/your/config.conf
   ```

## 基本键位绑定

mangowm 默认使用以下键位绑定：

| 键组合           | 动作                        |
| :--------------- | :-------------------------- |
| `Alt` + `Return` | 打开终端（默认为 `foot`）   |
| `Alt` + `Space`  | 打开启动器（默认为 `rofi`） |
| `Alt` + `Q`      | 关闭（杀死）活动窗口        |
| `Super` + `M`    | 退出 mangowm                |
| `Super` + `F`    | 切换全屏                    |
| `Alt` + `方向键` | 移动焦点（左、右、上、下）  |
| `Ctrl` + `1-9`   | 切换到标签 1-9              |
| `Alt` + `1-9`    | 将窗口移动到标签 1-9        |

> **警告：** 某些默认绑定依赖于特定工具，如 `foot`（终端）和 `rofi`（启动器）。确保你已安装它们，或在 `config.conf` 中更新为你喜欢的替代工具。

为了获得功能完整的桌面体验，我们推荐安装以下组件：

| 类别              | 推荐工具                                                           |
| :---------------- | :----------------------------------------------------------------- |
| 应用程序启动器    | rofi, bemenu, wmenu, fuzzel                                        |
| 终端模拟器        | foot, wezterm, alacritty, kitty, ghostty                           |
| 状态栏            | waybar, eww, quickshell, ags                                       |
| 桌面 Shell        | Noctalia, DankMaterialShell                                        |
| 壁纸设置          | swww, swaybg                                                       |
| 通知守护进程      | swaync, dunst, mako                                                |
| 桌面门户          | xdg-desktop-portal, xdg-desktop-portal-wlr, xdg-desktop-portal-gtk |
| 剪贴板            | wl-clipboard, wl-clip-persist, cliphist                            |
| Gamma 控制/夜间灯 | wlsunset, gammastep                                                |
| 杂项              | xfce-polkit, wlogout                                               |

```bash
git clone https://github.com/DreamMaoMao/mango-config.git ~/.config/mango
```
