## 设备配置

mangowm 提供对不同输入设备的细粒度控制。

### 键盘设置

控制按键重复率和布局规则。

| 设置                | 类型       | 默认值 | 描述                                                 |
| :------------------ | :--------- | :----- | :--------------------------------------------------- |
| `repeat_rate`       | `int`      | `25`   | 每秒按键重复次数。                                   |
| `repeat_delay`      | `int`      | `600`  | 按住按键后开始重复的延迟（毫秒）。                   |
| `numlockon`         | `0` 或 `1` | `0`    | 启动时启用 NumLock。                                 |
| `xkb_rules_rules`   | `string`   | -      | XKB 规则文件（例如 `evdev`、`base`）。通常自动检测。 |
| `xkb_rules_model`   | `string`   | -      | 键盘型号（例如 `pc104`、`macbook`）。                |
| `xkb_rules_layout`  | `string`   | -      | 键盘布局代码（例如 `us`、`de`、`us,de`）。           |
| `xkb_rules_variant` | `string`   | -      | 布局变体（例如 `dvorak`、`colemak`、`intl`）。       |
| `xkb_rules_options` | `string`   | -      | XKB 选项（例如 `caps:escape`、`ctrl:nocaps`）。      |

**示例：**

```ini
repeat_rate=40
repeat_delay=300
numlockon=1
xkb_rules_layout=us,de
xkb_rules_variant=dvorak
xkb_rules_options=caps:escape,ctrl:nocaps
```

### 触摸板设置

笔记本电脑触摸板的特定设置。某些设置可能需要重新登录才能生效。

| 设置                         | 默认值 | 描述                                    |
| :--------------------------- | :----- | :-------------------------------------- |
| `disable_trackpad`           | `0`    | 设置为 `1` 完全禁用触摸板。             |
| `tap_to_click`               | `1`    | 轻触触发左键点击。                      |
| `tap_and_drag`               | `1`    | 轻触并按住以拖动项目。                  |
| `trackpad_natural_scrolling` | `0`    | 反转滚动方向（自然滚动）。              |
| `scroll_method`              | `1`    | `1`（双指）、`2`（边缘）、`4`（按钮）。 |
| `click_method`               | `1`    | `1`（按钮区域）、`2`（点击手指）。      |
| `drag_lock`                  | `1`    | 轻触后锁定拖动。                        |
| `disable_while_typing`       | `1`    | 打字时禁用触摸板。                      |
| `left_handed`                | `0`    | 交换左/右键。                           |
| `middle_button_emulation`    | `0`    | 模拟中键。                              |
| `swipe_min_threshold`        | `1`    | 最小滑动手势阈值。                      |

---

**详细描述：**

- `scroll_method` 值：
  - `0` — 从不发送滚动事件（无滚动）。
  - `1` — 双指滚动：当两个手指在设备上逻辑按下时发送滚动事件。
  - `2` — 边缘滚动：当手指沿底部或右边缘移动时发送滚动事件。
  - `4` — 按钮滚动：当按住按钮且设备沿滚动轴移动时发送滚动事件。

- `click_method` 值：
  - `0` — 无软件点击模拟。
  - `1` — 按钮区域：在触摸板上使用软件定义的区域生成按钮事件。
  - `2` — 点击手指：手指数量决定按下哪个按钮。

- `accel_profile` 值：
  - `0` — 无加速。
  - `1` — 平面：无动态加速。指针速度 = 原始输入速度 × (1 + `accel_speed`)。
  - `2` — 自适应：慢速移动产生较少加速，快速移动产生更多加速。

- `button_map` 值：
  - `0` — 1/2/3 指轻触映射到左/右/中键。
  - `1` — 1/2/3 指轻触映射到左/中/右键。

- `send_events_mode` 值：
  - `0` — 正常从该设备发送事件。
  - `1` — 不从该设备发送事件。
  - `2` — 当插入外部指针设备时禁用该设备。

### 鼠标设置

外部鼠标的配置。

| 设置                      | 默认值 | 描述                                                  |
| :------------------------ | :----- | :---------------------------------------------------- |
| `mouse_natural_scrolling` | `0`    | 反转滚动方向。                                        |
| `accel_profile`           | `2`    | `0`（无）、`1`（平面）、`2`（自适应）。               |
| `accel_speed`             | `0.0`  | 速度调整（-1.0 到 1.0）。                             |
| `left_handed`             | `0`    | 交换左键和右键。                                      |
| `middle_button_emulation` | `0`    | 模拟中键。                                            |
| `swipe_min_threshold`     | `1`    | 最小滑动手势阈值。                                    |
| `send_events_mode`        | `0`    | `0`（启用）、`1`（禁用）、`2`（连接外部鼠标时禁用）。 |
| `button_map`              | `0`    | `0`（左/右/中），`1`（左/中/右）。                    |

## 键盘布局切换

要绑定多个布局并在它们之间切换，请在 `xkb_rules_layout` 中定义布局，并使用 `xkb_rules_options` 设置切换键组合。然后绑定 `switch_keyboard_layout` 来触发切换。

```ini
# 定义两个布局：US QWERTY 和 US Dvorak
xkb_rules_layout=us,us
xkb_rules_variant=,dvorak
xkb_rules_options=grp:lalt_lshift_toggle
```

或手动绑定到按键：

```ini
# 绑定 Alt+Shift_L 以循环切换键盘布局
bind=alt,shift_l,switch_keyboard_layout
```

随时使用 `mmsg -g -k` 查询当前键盘布局。

## 输入法编辑器 (IME)

要使用 Fcitx5 或 IBus，请在配置文件中设置这些环境变量。

> **信息：** 这些设置需要重启窗口管理器才能生效。

**Fcitx5：**

```ini
env=GTK_IM_MODULE,fcitx
env=QT_IM_MODULE,fcitx
env=QT_IM_MODULES,wayland;fcitx
env=SDL_IM_MODULE,fcitx
env=XMODIFIERS,@im=fcitx
env=GLFW_IM_MODULE,ibus
```

**IBus：**

```ini
env=GTK_IM_MODULE,ibus
env=QT_IM_MODULE,ibus
env=XMODIFIERS,@im=ibus
```
