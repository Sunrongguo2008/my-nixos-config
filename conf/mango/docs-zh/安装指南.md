## 软件包安装

mangowm 在多个发行版上提供预构建软件包。请选择你的发行版。

---

### AerynOS

mangowm 可在 **AerynOS 软件包仓库** 中获得。

你可以使用 `moss` 包管理器安装它：

```bash
sudo moss install mangowm
```

---

### Arch Linux

mangowm 可在 **Arch 用户仓库 (AUR)** 中获得。

你可以使用 AUR 助手（如 `yay` 或 `paru`）安装它：

```bash
yay -S mangowm-git
```

> **提示：** 此软件包拉取最新的 git 版本，确保你拥有最新的功能和修复。

---

### Fedora

该软件包位于第三方 **Terra 仓库** 中。首先添加 Terra 仓库。

> **警告：** 两个命令都需要 root 权限。如需要请使用 `sudo`。

```bash
dnf install --nogpgcheck --repofrompath 'terra,https://repos.fyralabs.com/terra$releasever' terra-release
```

然后安装软件包：

```bash
dnf install mangowm
```

---

### Gentoo

该软件包托管在社区维护的 **GURU** 仓库中。

1. **添加 GURU 仓库**
   ```bash
   emerge --ask --verbose eselect-repository
   eselect repository enable guru
   emerge --sync guru
   ```

2. **解除软件包屏蔽**
   将所需的软件包添加到你的 `package.accept_keywords` 文件中：
   - `gui-libs/scenefx`
   - `gui-wm/mangowm`

3. **安装 mango**
   ```bash
   emerge --ask --verbose gui-wm/mangowm
   ```

---

### Guix System

软件包定义在源代码仓库中描述。

1. **添加 mango channel**
   添加到 `$HOME/.config/guix/channels.scm`：
   ```scheme
   (cons (channel
           (name 'mangowm)
           (url "https://github.com/mangowm/mango.git")
           (branch "main"))
         %default-channels)
   ```

2. **安装**
   运行 `guix pull` 后，你可以安装 mangowm：
   ```bash
   guix install mangowm
   ```

   或使用 mangowm 模块将其添加到你的系统配置中：
   ```scheme
   (use-modules (mangowm))

   (packages (cons*
               mangowm-git
               ... ;; 其他软件包
               %base-packages))
   ```

> **提示：** 更多信息，请参阅 [Guix System 文档](https://guix.gnu.org/manual/devel/en/html_node/Channels.html)。

### NixOS

该仓库提供了一个带有 NixOS 模块的 Flake。

1. **添加 flake input**
   ```nix
   # flake.nix
   {
     inputs = {
       nixpkgs.url = "github:NixOS/nixpkgs/nixos-unstable";
       mangowm = {
         url = "github:mangowm/mango";
         inputs.nixpkgs.follows = "nixpkgs";
       };
       // 其他 inputs ...
     };
   }
   ```

2. **导入 NixOS 模块**

   **选项 A — 在 `configuration.nix` 中导入：**
   ```nix
   # configuration.nix（或你导入的任何其他文件）
   {inputs, ...}: {
     imports = [
       inputs.mangowm.nixosModules.mango
       // .. 其他导入 ...
     ];

     // ...
   }
   ```

   **选项 B — 直接在 flake 中导入：**
   ```nix
   # flake.nix
   {
     // ...

     outputs = { self, nixpkgs, mangowm, ...}@inputs: let
       inherit (nixpkgs) lib;
       // ...
     in {
       nixosConfigurations.YourHostName = lib.nixosSystem {
         modules = [
           mangowm.nixosModules.mango // 或 inputs.mangowm.nixosModules.mango
           // 其他导入 ...
         ];
       };
     }
   }
   ```

3. **启用模块**
   ```nix
   # configuration.nix（或你导入的任何其他文件）
   {
     programs.mango.enable = true;
   }
   ```

4. **额外选项**
   - `programs.mango.package` — 要使用的 mango 软件包，允许使用自定义 mango drvs
   - `programs.mango.addLoginEntry`（默认：`true`）— 添加登录条目到显示管理器

### PikaOS

mangowm 可在 **PikaOS 软件包仓库** 中获得。

你可以使用 `pikman` 包管理器安装它：

```bash

```

## 从源代码构建

如果你的发行版未在上面列出，或者你想要最新的未发布更改，你可以从源代码构建 mangowm。

> **信息：** 继续之前请确保已安装以下依赖：
> - `wayland`
> - `wayland-protocols`
> - `libinput`
> - `libdrm`
> - `libxkbcommon`
> - `pixman`
> - `libdisplay-info`
> - `libliftoff`
> - `hwdata`
> - `seatd`
> - `pcre2`
> - `xorg-xwayland`
> - `libxcb`

你还需要手动构建 `wlroots` 和 `scenefx`。

1. **构建 wlroots**
   克隆并安装所需的特定版本（查看 README 了解最新版本）。
   ```bash
   git clone -b 0.19.2 https://gitlab.freedesktop.org/wlroots/wlroots.git
   cd wlroots
   meson build -Dprefix=/usr
   sudo ninja -C build install
   ```

2. **构建 scenefx**
   此库处理视觉效果。
   ```bash
   git clone -b 0.4.1 https://github.com/wlrfx/scenefx.git
   cd scenefx
   meson build -Dprefix=/usr
   sudo ninja -C build install
   ```

3. **构建 mangowm**
   最后，编译合成器本身。
   ```bash
   git clone https://github.com/mangowm/mango.git
   cd mango
   meson build -Dprefix=/usr
   sudo ninja -C build install
   ```
