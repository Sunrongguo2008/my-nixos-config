## 概览设置

| 设置             | 类型 | 默认值 | 描述                                             |
| :--------------- | :--- | :----- | :----------------------------------------------- |
| `hotarea_size`   | 整数 | `10`   | 热区大小（像素）。                               |
| `enable_hotarea` | 整数 | `1`    | 启用热区（0：禁用，1：启用）。                   |
| `hotarea_corner` | 整数 | `2`    | 热区角落（0：左上，1：右上，2：左下，3：右下）。 |
| `ov_tab_mode`    | 整数 | `0`    | 概览标签页模式（0：禁用，1：启用）。             |
| `overviewgappih` | 整数 | `5`    | 概览模式内的内边距。                             |
| `overviewgappo`  | 整数 | `30`   | 概览模式的外边距。                               |

### 设置描述

- `enable_hotarea` — 当光标进入配置的角落时切换概览。
- `hotarea_size` — 热区触发区域的大小（像素）。
- `hotarea_corner` — 触发热区的角落（0：左上，1：右上，2：左下，3：右下）。
- `ov_tab_mode` — 在概览中循环聚焦窗口；释放修饰键退出概览。

### 概览中的鼠标交互

在概览模式下：

- **鼠标左键** — 跳转到（聚焦）窗口。
- **鼠标右键** — 关闭窗口。

要启用此行为，请在配置中添加以下鼠标绑定：

```ini
mousebind=NONE,btn_left,toggleoverview,1
mousebind=NONE,btn_right,killclient,0
```
