## 模糊 (Blur)

模糊为透明窗口创建毛玻璃效果。

| 设置                     | 默认值 | 描述                                                                                                     |
| :----------------------- | :----- | :------------------------------------------------------------------------------------------------------- |
| `blur`                   | `0`    | 为窗口启用模糊效果。                                                                                     |
| `blur_layer`             | `0`    | 为 Layer 表面（如状态栏/停靠栏）启用模糊。                                                               |
| `blur_optimized`         | `1`    | 缓存壁纸和模糊背景，显著降低 GPU 使用率。禁用它会显著增加 GPU 消耗并可能导致渲染卡顿。**强烈推荐启用。** |
| `blur_params_radius`     | `5`    | 模糊强度（半径）。                                                                                       |
| `blur_params_num_passes` | `1`    | passes 数量。数值越高 = 越平滑但消耗越大。                                                               |
| `blur_params_noise`      | `0.02` | 模糊噪点级别。                                                                                           |
| `blur_params_brightness` | `0.9`  | 模糊亮度调整。                                                                                           |
| `blur_params_contrast`   | `0.9`  | 模糊对比度调整。                                                                                         |
| `blur_params_saturation` | `1.2`  | 模糊饱和度调整。                                                                                         |

> **警告：** 模糊对性能影响相对较大。如果你的硬件有限，不建议启用。如果启用模糊后遇到卡顿，请确保 `blur_optimized=1` —— 禁用它会显著增加 GPU 消耗并可能导致渲染卡顿。要完全禁用模糊，设置 `blur=0`。

## 阴影 (Shadows)

投影有助于区分浮动窗口和背景。

| 设置                   | 默认值       | 描述                               |
| :--------------------- | :----------- | :--------------------------------- |
| `shadows`              | `0`          | 启用阴影。                         |
| `layer_shadows`        | `0`          | 为 Layer 表面启用阴影。            |
| `shadow_only_floating` | `1`          | 仅为浮动窗口绘制阴影（节省性能）。 |
| `shadows_size`         | `10`         | 阴影大小。                         |
| `shadows_blur`         | `15`         | 阴影模糊程度。                     |
| `shadows_position_x`   | `0`          | 阴影 X 偏移。                      |
| `shadows_position_y`   | `0`          | 阴影 Y 偏移。                      |
| `shadowscolor`         | `0x000000ff` | 阴影颜色。                         |

```ini
# 阴影配置示例
shadows=1
layer_shadows=1
shadow_only_floating=1
shadows_size=12
shadows_blur=15
shadows_position_x=0
shadows_position_y=0
shadowscolor=0x000000ff
```

## 不透明度与圆角 (Opacity & Corner Radius)

控制窗口的透明度和圆角。

| 设置                             | 默认值 | 描述                                                                                         |
| :------------------------------- | :----- | :------------------------------------------------------------------------------------------- |
| `border_radius`                  | `0`    | 窗口圆角半径（像素）。                                                                       |
| `border_radius_location_default` | `0`    | 圆角位置：`0`（全部），`1`（左上），`2`（右上），`3`（左下），`4`（右下），`5`（最近的角）。 |
| `no_radius_when_single`          | `0`    | 如果只有一个窗口可见则禁用圆角。                                                             |
| `focused_opacity`                | `1.0`  | 活动窗口的不透明度（0.0 - 1.0）。                                                            |
| `unfocused_opacity`              | `1.0`  | 非活动窗口的不透明度（0.0 - 1.0）。                                                          |

```ini
# 窗口圆角半径（像素）
border_radius=0

# 圆角位置（0=全部，1=左上，2=右上，3=左下，4=右下）
border_radius_location_default=0

# 如果只有一个窗口可见则禁用圆角
no_radius_when_single=0

# 活动窗口的不透明度（0.0 - 1.0）
focused_opacity=1.0

# 非活动窗口的不透明度
unfocused_opacity=1.0
```
