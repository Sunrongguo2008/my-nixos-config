## 尺寸

控制窗口边框和间隙的大小。

| 设置       | 默认值 | 描述                               |
| :--------- | :----- | :--------------------------------- |
| `borderpx` | `4`    | 边框宽度（像素）。                 |
| `gappih`   | `5`    | 水平内边距（窗口之间）。           |
| `gappiv`   | `5`    | 垂直内边距。                       |
| `gappoh`   | `10`   | 水平外边距（窗口和屏幕边缘之间）。 |
| `gappov`   | `10`   | 垂直外边距。                       |

## 颜色

颜色使用 `0xRRGGBBAA` 十六进制格式定义。

```ini
# 根窗口的背景颜色
rootcolor=0x323232ff

# 非活动窗口边框
bordercolor=0x444444ff

# 活动窗口边框
focuscolor=0xc66b25ff

# 紧急窗口边框（警报）
urgentcolor=0xad401fff
```

### 特定状态颜色

你还可以根据窗口状态进行颜色编码：

| 状态       | 配置键                | 默认颜色     |
| :--------- | :-------------------- | :----------- |
| 最大化     | `maximizescreencolor` | `0x89aa61ff` |
| Scratchpad | `scratchpadcolor`     | `0x516c93ff` |
| 全局       | `globalcolor`         | `0xb153a7ff` |
| 覆盖       | `overlaycolor`        | `0x14a57cff` |

> **提示：** 有关 scratchpad 窗口大小设置，请参阅 [Scratchpad](/docs/window-management/scratchpad) 配置。

## 光标主题

设置鼠标光标的大小和主题。

```ini
cursor_size=24
cursor_theme=Adwaita
```

> **提示：** 你可能还想设置 `XCURSOR_SIZE` 环境变量以匹配：
> `env=XCURSOR_SIZE,24`
