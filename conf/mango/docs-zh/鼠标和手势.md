## 鼠标绑定

将动作分配给鼠标按钮按下，可选修饰键。

### 语法

```ini
mousebind=MODIFIERS,BUTTON,COMMAND,PARAMETERS
```

- **修饰键**：`SUPER`、`CTRL`、`ALT`、`SHIFT`、`NONE`。使用 `+` 组合（例如 `SUPER+CTRL`）
- **按钮**：`btn_left`、`btn_right`、`btn_middle`、`btn_side`、`btn_extra`、`btn_forward`、`btn_back`、`btn_task`

> **警告：** 当修饰键设置为 `NONE` 时，只有 `btn_middle` 在普通模式下工作。`btn_left` 和 `btn_right` 仅在概览模式下工作。

### 示例

```ini
# 窗口操作
mousebind=SUPER,btn_left,moveresize,curmove
mousebind=SUPER,btn_right,moveresize,curresize
mousebind=SUPER+CTRL,btn_right,killclient

# 概览模式（需要 NONE 修饰键）
mousebind=NONE,btn_left,toggleoverview,-1
mousebind=NONE,btn_right,killclient,0
mousebind=NONE,btn_middle,togglemaximizescreen,0
```

## 轴绑定

将滚轮滚动映射到工作区和窗口导航的动作。

### 语法

```ini
axisbind=MODIFIERS,DIRECTION,COMMAND,PARAMETERS
```

- **方向**：`UP`、`DOWN`、`LEFT`、`RIGHT`

### 示例

```ini
axisbind=SUPER,UP,viewtoleft_have_client
axisbind=SUPER,DOWN,viewtoright_have_client
```

## 手势绑定

启用触摸板滑动手势用于导航和窗口管理。

### 语法

```ini
gesturebind=MODIFIERS,DIRECTION,FINGERS,COMMAND,PARAMETERS
```

- **方向**：`up`、`down`、`left`、`right`
- **手指**：`3` 或 `4`

> **信息：** 手势需要正确的触摸板配置。有关触摸板设置（如 `tap_to_click` 和 `disable_while_typing`），请参阅 [输入设备](/docs/configuration/input)。

### 示例

```ini
# 3 指：窗口焦点
gesturebind=none,left,3,focusdir,left
gesturebind=none,right,3,focusdir,right
gesturebind=none,up,3,focusdir,up
gesturebind=none,down,3,focusdir,down

# 4 指：工作区导航
gesturebind=none,left,4,viewtoleft_have_client
gesturebind=none,right,4,viewtoright_have_client
gesturebind=none,up,4,toggleoverview
gesturebind=none,down,4,toggleoverview
```

## 开关绑定

在硬件事件（如笔记本盖子打开/关闭）上触发动作。

### 语法

```ini
switchbind=FOLD_STATE,COMMAND,PARAMETERS
```

- **折叠状态**：`fold`（盖子关闭）、`unfold`（盖子打开）

> **警告：** 在 `/etc/systemd/logind.conf` 中禁用系统盖子处理：
>
> ```ini
> HandleLidSwitch=ignore
> HandleLidSwitchExternalPower=ignore
> HandleLidSwitchDocked=ignore
> ```

### 示例

```ini
switchbind=fold,spawn,swaylock -f -c 000000
switchbind=unfold,spawn,wlr-dpms on
```
