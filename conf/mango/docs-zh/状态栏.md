## 模块配置

mangowm 兼容 Waybar 的 `ext/workspaces` 模块（Wayland 标准）或 `dwl/tags` 模块。我们推荐 `ext/workspaces` 以获得最佳体验。

> **提示：** 你也可以使用 `dwl/tags` 模块，但 `ext/workspaces` 提供更好的 mangowm 功能集成。`ext/workspaces` 模块需要 **Waybar > 0.14.0**。

### `config.jsonc`

将以下内容添加到你的 Waybar 配置中：

```jsonc
{
  "modules-left": [
    "ext/workspaces",
    "dwl/window"
  ],
  "ext/workspaces": {
    "format": "{icon}",
    "ignore-hidden": true,
    "on-click": "activate",
    "on-click-right": "deactivate",
    "sort-by-id": true
  },
  "dwl/window": {
    "format": "[{layout}] {title}"
  }
}
```

你可以使用 `style.css` 中的标准 CSS 来设置标签样式。

  border-style: solid;
  border-color: #c9b890;
  margin-left: 4px;
  padding-left: 10px;
  padding-right: 6px;
  background: rgba(40, 40, 40, 0.76);
}
