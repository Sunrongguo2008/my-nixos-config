---
title: 动画 (Animations)
description: 为窗口和图层配置平滑过渡效果。
---

## 启用动画

mangowm 支持标准窗口和 Layer Shell 表面（如状态栏和通知）的动画效果。

```ini
animations=1
layer_animations=1
```

## 动画类型

你可以为窗口和图层表面的打开和关闭定义不同的动画样式。

可用类型：`slide`（滑动）、`zoom`（缩放）、`fade`（淡入淡出）、`none`（无）。

```ini
animation_type_open=zoom
animation_type_close=slide
layer_animation_type_open=slide
layer_animation_type_close=slide
```

## 淡入淡出设置

控制动画的淡入和淡出效果。

```ini
animation_fade_in=1
animation_fade_out=1
fadein_begin_opacity=0.5
fadeout_begin_opacity=0.5
```

- `animation_fade_in` — 启用淡入效果（0：禁用，1：启用）
- `animation_fade_out` — 启用淡出效果（0：禁用，1：启用）
- `fadein_begin_opacity` — 淡入动画的起始不透明度（0.0–1.0）
- `fadeout_begin_opacity` — 淡出动画的起始不透明度（0.0–1.0）

## 缩放设置

调整缩放动画的缩放比例。

```ini
zoom_initial_ratio=0.3
zoom_end_ratio=0.8
```

- `zoom_initial_ratio` — 初始缩放比例
- `zoom_end_ratio` — 结束缩放比例

## 持续时间

控制动画速度（以毫秒为单位）。

| 设置                       | 类型 | 默认值 | 描述                                         |
| :------------------------- | :--- | :----- | :------------------------------------------- |
| `animation_duration_move`  | 整数 | `500`  | 移动动画持续时间（毫秒）                     |
| `animation_duration_open`  | 整数 | `400`  | 打开动画持续时间（毫秒）                     |
| `animation_duration_tag`   | 整数 | `300`  | 标签切换动画持续时间（毫秒）                 |
| `animation_duration_close` | 整数 | `300`  | 关闭动画持续时间（毫秒）                     |
| `animation_duration_focus` | 整数 | `0`    | 焦点切换（不透明度过渡）动画持续时间（毫秒） |

```ini
animation_duration_move=500
animation_duration_open=400
animation_duration_tag=300
animation_duration_close=300
animation_duration_focus=0
```

## 自定义贝塞尔曲线

贝塞尔曲线决定动画的"感觉"（例如线性或有弹性）。格式为 `x1,y1,x2,y2`。

你可以使用在线工具（如 [cssportal.com](https://www.cssportal.com/css-cubic-bezier-generator/) 或 [easings.net](https://easings.net)）来可视化和生成曲线值。

| 设置                         | 类型   | 默认值               | 描述                                   |
| :--------------------------- | :----- | :------------------- | :------------------------------------- |
| `animation_curve_open`       | 字符串 | `0.46,1.0,0.29,0.99` | 打开动画贝塞尔曲线                     |
| `animation_curve_move`       | 字符串 | `0.46,1.0,0.29,0.99` | 移动动画贝塞尔曲线                     |
| `animation_curve_tag`        | 字符串 | `0.46,1.0,0.29,0.99` | 标签动画贝塞尔曲线                     |
| `animation_curve_close`      | 字符串 | `0.46,1.0,0.29,0.99` | 关闭动画贝塞尔曲线                     |
| `animation_curve_focus`      | 字符串 | `0.46,1.0,0.29,0.99` | 焦点切换（不透明度过渡）动画贝塞尔曲线 |
| `animation_curve_opafadein`  | 字符串 | `0.46,1.0,0.29,0.99` | 打开不透明度动画贝塞尔曲线             |
| `animation_curve_opafadeout` | 字符串 | `0.5,0.5,0.5,0.5`    | 关闭不透明度动画贝塞尔曲线             |

```ini
animation_curve_open=0.46,1.0,0.29,0.99
animation_curve_move=0.46,1.0,0.29,0.99
animation_curve_tag=0.46,1.0,0.29,0.99
animation_curve_close=0.46,1.0,0.29,0.99
animation_curve_focus=0.46,1.0,0.29,0.99
animation_curve_opafadein=0.46,1.0,0.29,0.99
animation_curve_opafadeout=0.5,0.5,0.5,0.5
```

## 标签动画方向

控制标签切换动画的方向。

| 设置                      | 默认值 | 描述                             |
| :------------------------ | :----- | :------------------------------- |
| `tag_animation_direction` | `1`    | 标签动画方向（1：水平，0：垂直） |
