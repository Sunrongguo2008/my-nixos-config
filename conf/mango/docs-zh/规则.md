## 窗口规则

窗口规则允许你根据应用程序的 `appid` 或 `title` 为应用程序设置特定属性（浮动、不透明度、大小、动画等）。你可以在一行中设置所有参数，如果同时设置了 appid 和 title，则只有当 appid 和 title 都匹配时窗口才会遵循规则。

**格式：**

```ini
windowrule=Parameter:Values,title:Values
windowrule=Parameter:Values,Parameter:Values,appid:Values,title:Values
```

### 状态和行为参数

| 参数                      | 类型   | 值                  | 描述                                             |
| :------------------------ | :----- | :------------------ | :----------------------------------------------- |
| `appid`                   | 字符串 | 任意                | 按应用程序 ID 匹配，支持正则表达式               |
| `title`                   | 字符串 | 任意                | 按窗口标题匹配，支持正则表达式                   |
| `isfloating`              | 整数   | `0` / `1`           | 强制浮动状态                                     |
| `isfullscreen`            | 整数   | `0` / `1`           | 强制全屏状态                                     |
| `isfakefullscreen`        | 整数   | `0` / `1`           | 强制伪全屏状态（窗口保持受限）                   |
| `isglobal`                | 整数   | `0` / `1`           | 作为全局窗口打开（在所有标签上粘性）             |
| `isoverlay`               | 整数   | `0` / `1`           | 使其始终在顶层                                   |
| `isopensilent`            | 整数   | `0` / `1`           | 打开时不聚焦                                     |
| `istagsilent`             | 整数   | `0` / `1`           | 如果客户端不在当前视图标签中则不聚焦             |
| `force_maximize`          | 整数   | `0` / `1`（默认 1） | 客户端状态默认为最大化                           |
| `ignore_maximize`         | 整数   | `0` / `1`（默认 1） | 不处理客户端的最大化请求                         |
| `ignore_minimize`         | 整数   | `0` / `1`（默认 1） | 不处理客户端的最小化请求                         |
| `force_tiled_state`       | 整数   | `0` / `1`           | 欺骗窗口认为它是平铺的，以便更好地遵守分配的维度 |
| `noopenmaximized`         | 整数   | `0` / `1`           | 窗口不以最大化模式打开                           |
| `single_scratchpad`       | 整数   | `0` / `1`（默认 1） | 只显示一个命名 scratchpad 或普通 scratchpad      |
| `allow_shortcuts_inhibit` | 整数   | `0` / `1`（默认 1） | 允许客户端禁用快捷键                             |
| `indleinhibit_when_focus` | 整数   | `0` / `1`（默认 0） | 当此窗口聚焦时自动保持空闲禁用激活               |

### 几何和位置

| 参数              | 类型   | 值        | 描述                                                                                                             |
| :---------------- | :----- | :-------- | :--------------------------------------------------------------------------------------------------------------- |
| `width`           | 整数   | 0-9999    | 窗口成为浮动窗口时的宽度                                                                                         |
| `height`          | 整数   | 0-9999    | 窗口成为浮动窗口时的高度                                                                                         |
| `offsetx`         | 整数   | -999-999  | 距中心的 X 偏移（%），100 是带外边距的屏幕边缘                                                                   |
| `offsety`         | 整数   | -999-999  | 距中心的 Y 偏移（%），100 是带外边距的屏幕边缘                                                                   |
| `monitor`         | 字符串 | 任意      | 通过 [monitor spec](/docs/configuration/monitors#monitor-spec-format) 分配到显示器（名称、制造商、型号或序列号） |
| `tags`            | 整数   | 1-9       | 分配到特定标签                                                                                                   |
| `no_force_center` | 整数   | `0` / `1` | 窗口不强制居中                                                                                                   |
| `isnosizehint`    | 整数   | `0` / `1` | 不使用最小大小和最大大小作为大小提示                                                                             |

### 视觉效果和装饰

| 参数                | 类型 | 值        | 描述                 |
| :------------------ | :--- | :-------- | :------------------- |
| `noblur`            | 整数 | `0` / `1` | 窗口无模糊效果       |
| `isnoborder`        | 整数 | `0` / `1` | 移除窗口边框         |
| `isnoshadow`        | 整数 | `0` / `1` | 不应用阴影           |
| `isnoradius`        | 整数 | `0` / `1` | 不应用圆角           |
| `isnoanimation`     | 整数 | `0` / `1` | 不应用动画           |
| `focused_opacity`   | 整数 | `0` / `1` | 窗口聚焦时不透明度   |
| `unfocused_opacity` | 整数 | `0` / `1` | 窗口未聚焦时不透明度 |
| `allow_csd`         | 整数 | `0` / `1` | 允许客户端侧装饰     |

> **提示：** 有关详细的视觉效果配置，请参阅 [窗口效果](/docs/visuals/effects) 页面了解模糊、阴影和不透明度设置。

### 布局和 Scroller

| 参数                         | 类型   | 值      | 描述                                 |
| :--------------------------- | :----- | :------ | :----------------------------------- |
| `scroller_proportion`        | 浮点数 | 0.1-1.0 | 设置 scroller 比例                   |
| `scroller_proportion_single` | 浮点数 | 0.1-1.0 | 设置 scroller 单个窗口时自动调整比例 |

> **提示：** 有关全面的布局配置，请参阅 [布局](/docs/window-management/layouts) 页面了解所有布局选项和详细设置。

### 动画

| 参数                   | 类型   | 值                      | 描述             |
| :--------------------- | :----- | :---------------------- | :--------------- |
| `animation_type_open`  | 字符串 | zoom, slide, fade, none | 设置打开动画     |
| `animation_type_close` | 字符串 | zoom, slide, fade, none | 设置关闭动画     |
| `nofadein`             | 整数   | `0` / `1`               | 窗口忽略淡入动画 |
| `nofadeout`            | 整数   | `0` / `1`               | 窗口忽略淡出动画 |

> **提示：** 有关详细的动画配置，请参阅 [动画](/docs/visuals/animations) 页面了解可用类型和设置。

### 终端和吞咽

| 参数        | 类型 | 值        | 描述                                    |
| :---------- | :--- | :-------- | :-------------------------------------- |
| `isterm`    | 整数 | `0` / `1` | 当打开时新的 GUI 窗口将替换 isterm 窗口 |
| `noswallow` | 整数 | `0` / `1` | 窗口不会替换 isterm 窗口                |

### 全局和特殊窗口

| 参数                | 类型   | 值                          | 描述                                                 |
| :------------------ | :----- | :-------------------------- | :--------------------------------------------------- |
| `globalkeybinding`  | 字符串 | `[mod combination][-][key]` | 全局键绑定（仅适用于 Wayland 应用程序）              |
| `isunglobal`        | 整数   | `0` / `1`                   | 作为未管理的全局窗口打开（用于桌面宠物或摄像头窗口） |
| `isnamedscratchpad` | 整数   | `0` / `1`                   | 0：禁用，1：命名 scratchpad                          |

> **提示：** 有关 scratchpad 用法，请参阅 [Scratchpad](/docs/window-management/scratchpad) 页面了解详细的配置示例。

### 性能和撕裂

| 参数            | 类型 | 值        | 描述                                                                              |
| :-------------- | :--- | :-------- | :-------------------------------------------------------------------------------- |
| `force_tearing` | 整数 | `0` / `1` | 将窗口设置为撕裂状态，参考 [撕裂](/docs/configuration/monitors#tearing-game-mode) |

### 示例

```ini
# 设置特定窗口大小和位置
windowrule=width:1000,height:900,appid:yesplaymusic,title:Demons

# OBS Studio 的全局键绑定
windowrule=globalkeybinding:ctrl+alt-o,appid:com.obsproject.Studio
windowrule=globalkeybinding:ctrl+alt-n,appid:com.obsproject.Studio
windowrule=isopensilent:1,appid:com.obsproject.Studio

# 为游戏强制撕裂
windowrule=force_tearing:1,title:vkcube
windowrule=force_tearing:1,title:Counter-Strike 2

# 文件管理器的命名 scratchpad
windowrule=isnamedscratchpad:1,width:1280,height:800,appid:st-yazi

# 特定应用的自定义不透明度
windowrule=focused_opacity:0.8,appid:firefox
windowrule=unfocused_opacity:0.6,appid:foot

# 为选择工具禁用模糊
windowrule=noblur:1,appid:slurp

# 相对于屏幕中心定位窗口
windowrule=offsetx:20,offsety:-30,width:800,height:600,appid:alacritty

# 发送到特定标签和显示器
windowrule=tags:9,monitor:HDMI-A-1,appid:discord

# 终端吞咽设置
windowrule=isterm:1,appid:st
windowrule=noswallow:1,appid:foot

# 禁用客户端侧装饰
windowrule=allow_csd:1,appid:firefox

# 未管理的全局窗口（桌面宠物、摄像头）
windowrule=isunglobal:1,appid:cheese

# 命名 scratchpad 切换
bind=alt,h,toggle_named_scratchpad,st-yazi,none,st -c st-yazi -e yazi
```

## 标签规则

你可以在一行中设置所有参数。如果只设置了 `id`，则当 id 匹配时遵循规则。如果设置了 `monitor_name`、`monitor_make`、`monitor_model` 或 `monitor_serial` 中的任何字段，则只有当**所有**设置的显示器字段都匹配时才遵循规则。

> **警告：** 标签规则中设置的布局优先级高于显示器规则布局。

**格式：**

```ini
tagrule=id:Values,Parameter:Values,Parameter:Values
tagrule=id:Values,monitor_name:eDP-1,Parameter:Values,Parameter:Values
tagrule=id:Values,monitor_make:xxx,monitor_model:xxx,Parameter:Values
```

> **提示：** 有关每种布局类型的详细描述，请参阅 [布局](/docs/window-management/layouts#supported-layouts)。

| 参数               | 类型   | 值           | 描述                           |
| :----------------- | :----- | :----------- | :----------------------------- |
| `id`               | 整数   | 0-9          | 按标签 id 匹配，0 表示 ~0 标签 |
| `monitor_name`     | 字符串 | 显示器名称   | 按显示器名称匹配               |
| `monitor_make`     | 字符串 | 显示器制造商 | 按显示器制造商匹配             |
| `monitor_model`    | 字符串 | 显示器型号   | 按显示器型号匹配               |
| `monitor_serial`   | 字符串 | 显示器序列号 | 按显示器序列号匹配             |
| `layout_name`      | 字符串 | 布局名称     | 要设置的布局名称               |
| `no_render_border` | 整数   | `0` / `1`    | 禁用渲染边框                   |
| `no_hide`          | 整数   | `0` / `1`    | 即使标签为空也不隐藏           |
| `nmaster`          | 整数   | 0, 99        | master 窗口数量                |
| `mfact`            | 浮点数 | 0.1–0.9      | master 区域因子                |

### 示例

```ini
# 为特定标签设置布局
tagrule=id:1,layout_name:scroller
tagrule=id:2,layout_name:scroller

# 限制到特定显示器
tagrule=id:1,monitor_name:eDP-1,layout_name:scroller
tagrule=id:2,monitor_name:eDP-1,layout_name:scroller

# 持久标签（1-4）带布局分配
tagrule=id:1,no_hide:1,layout_name:scroller
tagrule=id:2,no_hide:1,layout_name:scroller
tagrule=id:3,monitor_name:eDP-1,no_hide:1,layout_name:scroller
tagrule=id:4,monitor_name:eDP-1,no_hide:1,layout_name:scroller

# 带 master 布局设置的高级标签配置
tagrule=id:5,layout_name:tile,nmaster:2,mfact:0.6
tagrule=id:6,monitor_name:HDMI-A-1,layout_name:monocle,no_render_border:1
```

> **提示：** 有关 Waybar 配置和持久标签，请参阅 [状态栏](/docs/visuals/status-bar) 文档。

## 图层规则

你可以在一行中设置所有参数。目标"layer shell"表面，如状态栏（`waybar`）、启动器（`rofi`）或通知守护进程。

**格式：**

```ini
layerrule=layer_name:Values,Parameter:Values,Parameter:Values
```

> **提示：** 你可以使用 `mmsg -e` 获取最后打开的图层名称用于调试。

| 参数                   | 类型   | 值                      | 描述                           |
| :--------------------- | :----- | :---------------------- | :----------------------------- |
| `layer_name`           | 字符串 | 图层名称                | 按图层名称匹配，支持正则表达式 |
| `animation_type_open`  | 字符串 | slide, zoom, fade, none | 设置打开动画                   |
| `animation_type_close` | 字符串 | slide, zoom, fade, none | 设置关闭动画                   |
| `noblur`               | 整数   | `0` / `1`               | 禁用模糊                       |
| `noanim`               | 整数   | `0` / `1`               | 禁用图层动画                   |
| `noshadow`             | 整数   | `0` / `1`               | 禁用图层阴影                   |

> **提示：** 有关动画类型，请参阅 [动画](/docs/visuals/animations#animation-types)。有关视觉效果，请参阅 [窗口效果](/docs/visuals/effects)。

### 示例

```ini
# slurp 选择图层无模糊或动画（避免截图中的遮挡和重影）
layerrule=noanim:1,noblur:1,layer_name:selection

# Rofi 的缩放动画带多个参数
layerrule=animation_type_open:zoom,noanim:0,layer_name:rofi

# 为通知守护进程禁用动画和阴影
layerrule=noanim:1,noshadow:1,layer_name:swaync

# 启动器的多种效果
layerrule=animation_type_open:slide,animation_type_close:fade,noblur:1,layer_name:wofi
```
