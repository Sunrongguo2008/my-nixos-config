## 系统和硬件

| 设置                      | 默认值 | 描述                                                                     |
| :------------------------ | :----- | :----------------------------------------------------------------------- |
| `xwayland_persistence`    | `1`    | 即使没有 X11 应用程序打开也让 XWayland 保持运行（减少启动延迟）。        |
| `syncobj_enable`          | `0`    | 启用 `drm_syncobj` 时间线支持（有助于减少游戏卡顿/延迟）。**需要重启。** |
| `allow_lock_transparent`  | `0`    | 允许锁屏透明。                                                           |
| `allow_shortcuts_inhibit` | `1`    | 允许客户端禁用快捷键。                                                   |
| `vrr`                     | -      | 通过 [显示器规则](/docs/configuration/monitors#monitor-rules) 设置。     |

## 焦点和输入

| 设置                      | 默认值 | 描述                                              |
| :------------------------ | :----- | :------------------------------------------------ |
| `focus_on_activate`       | `1`    | 当窗口请求激活时自动聚焦。                        |
| `sloppyfocus`             | `1`    | 焦点跟随鼠标光标。                                |
| `warpcursor`              | `1`    | 当通过键盘改变焦点时将光标移动到窗口中心。        |
| `cursor_hide_timeout`     | `0`    | 在 N 秒无活动后隐藏光标（`0` 禁用）。             |
| `drag_tile_to_tile`       | `0`    | 允许将一个平铺窗口拖放到另一个窗口上来交换位置。  |
| `drag_corner`             | `3`    | 拖放平铺检测的角（0：无，1–3：角，4：自动检测）。 |
| `drag_warp_cursor`        | `1`    | 拖放窗口平铺时 warp 光标。                        |
| `axis_bind_apply_timeout` | `100`  | 检测连续滚动事件以进行轴绑定的超时时间（毫秒）。  |
| `axis_scroll_factor`      | `1.0`  | 轴滚动速度的滚动因子（0.1–10.0）。                |

## 多显示器和标签

| 设置                       | 默认值 | 描述                                                                      |
| :------------------------- | :----- | :------------------------------------------------------------------------ |
| `focus_cross_monitor`      | `0`    | 允许方向性焦点跨越显示器边界。                                            |
| `exchange_cross_monitor`   | `0`    | 允许跨越显示器边界交换客户端。                                            |
| `focus_cross_tag`          | `0`    | 允许方向性焦点跨越到其他标签。                                            |
| `view_current_to_back`     | `0`    | 切换当前标签时切换回之前查看的标签。                                      |
| `scratchpad_cross_monitor` | `0`    | 在所有显示器之间共享 scratchpad 池。                                      |
| `single_scratchpad`        | `1`    | 只允许一个 scratchpad（命名或标准）同时可见。                             |
| `circle_layout`            | -      | `switch_layout` 循环遍历的布局列表（逗号分隔），示例值：`tile,scroller`。 |

## 窗口行为

| 设置                             | 默认值 | 描述                                                                     |
| :------------------------------- | :----- | :----------------------------------------------------------------------- |
| `enable_floating_snap`           | `0`    | 将浮动窗口捕捉到边缘或其他窗口。                                         |
| `snap_distance`                  | `30`   | 触发浮动捕捉的最大距离（像素）。                                         |
| `no_border_when_single`          | `0`    | 当标签上只有一个窗口可见时移除窗口边框。                                 |
| `idleinhibit_ignore_visible`     | `0`    | 允许不可见的客户端（例如后台音频播放器）禁用空闲。                       |
| `drag_tile_refresh_interval`     | `8.0`  | 拖放时刷新平铺窗口调整大小的间隔（1.0–16.0）。太小可能导致应用程序卡顿。 |
| `drag_floating_refresh_interval` | `8.0`  | 拖放时刷新浮动窗口调整大小的间隔（1.0–16.0）。太小可能导致应用程序卡顿。 |
